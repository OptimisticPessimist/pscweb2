# pscweb2

## Repository

- [https://github.com/satamame/pscweb2](https://github.com/satamame/pscweb2)

## Overview

- 演劇製作を少しでも楽にすることを目的として、手探りで開発中のウェブアプリです。
- 稽古の進捗やスケジュールを管理します。
- 稽古プランを立てやすくするため、シーンごとにスケジュールを可視化します。
- ゆくゆくは台本データをアップロードしてシーンを生成したりしたいです。
